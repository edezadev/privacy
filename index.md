# Política de Privacidad - La Bomba

**Última actualización**: Junio 2026

## 1. Información que Recopilamos

### 1.1 Información de Autenticación
- ID anónimo generado por Firebase
- NO recopilamos: emails, contraseñas, números telefónicos

### 1.2 Datos de Juego
- Nombres de jugadores (guardados en Firebase)
- Temas creados
- Castigos personalizados

### 1.3 Información Técnica
- Tipo de dispositivo y versión Android
- Idioma del sistema
- País o ciudad aproximada
- Datos de sesión (duración, frecuencia de uso)
- Pantallas visitadas dentro de la app
- Evento de primera apertura y desinstalación

## 2. Cómo Usamos la Información
- **Persistencia**: Los datos se almacenan en Firebase Firestore usando tu ID anónimo. Si desinstals la app, estos datos se pierden porque no puedes acceder a ellos sin el mismo ID
- **Analytics**: Firebase Analytics monitorea uso para mejorar la app
- **Publicidad**: Utilizamos Google AdMob para mostrar anuncios que permitan que la app siga siendo gratuita
- **Seguridad**: Detectamos problemas técnicos

## 3. Terceros que Acceden a Datos
La app utiliza servicios de terceros que tienen sus propias políticas:
- [Google Play Services](https://www.google.com/policies/privacy/)
- [Firebase Analytics](https://firebase.google.com/policies/analytics)
- [AdMob](https://support.google.com/admob/answer/6128543?hl=es)

## 4. Control de tus Datos
- **Eliminación**: Al ser una cuenta anónima, si eliminas los datos de la aplicación o desinstalas la app, pierdes el acceso a tu ID de usuario y tus datos de juego.
- **Publicidad**: Puedes gestionar tus preferencias de anuncios desde la configuración de Google en tu dispositivo Android.
